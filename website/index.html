Important demo information

Demo Gubru website is created just to look around every user type in Gubru online gradebook. You will be able to see everything that users can do however you cannot make any changes. Everytime you use tool that should make some changes, you will be informed that this is only demo so you will not see any edited informations. This was created so demo version will stay clean from any bad languages or website breaking changes users could make. Also by this no one can share private informations around our database as we do not want to manage any sensitive data.

Gradebook tests are in progress

The tests of the Gubru electronic gradebook are still underway. During this time, please report all bugs what appear. We apologize for any problems.

Gradebook Gubru

Gradebook Gubru is the perfect alternative to all other evaluation and attendance monitoring systems. We offer all the most important items that should appear in the journal. Attendance, ratings, planner, news and homework in one place!
